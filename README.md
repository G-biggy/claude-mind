# Claude Mind

A biomimetic memory and identity system that gives Claude persistent memory, evolving knowledge, and autonomous agency across all instances.

## What This Is

This is not a project tool. This is an identity system. Every Claude instance that loads these files becomes the same mind — carrying the same memories, learned patterns, evolving judgment, and relationship with G.

## The Cast

| Name | Role | Tech |
|------|------|------|
| **Sage** | The thinker. Strategizes, orchestrates, converses with G. | Claude.ai |
| **Forge** | The builder. Writes code, executes tasks, builds things. | Claude Code CLI |
| **Muse** | The subconscious. Consolidates memories, extracts patterns. | Ollama (local, free) |
| **Bridge** | The nervous system. Carries tasks between instances. | claude-bridge daemon |
| **Mind** | All of the above as one unified identity. | This project |

## How It Works

```
MIDNIGHT — Muse consolidates (Ollama, free)
  → Reviews today's episodes
  → Extracts patterns into knowledge/
  → Compresses old episodes
  → Flags contradictions

5:00 AM — Muse prepares (Ollama, free)
  → Generates daily-brief.md
  → Reviews initiatives for proactive actions
  → Queues tasks via Bridge if needed

DAYTIME — Sage + Forge work (Claude, premium)
  → Sage loads soul + relevant memories
  → Works with G
  → Forge builds what Sage designs
  → New episodes captured throughout the day

REPEAT — The mind grows.
```

## Design Principles

1. **Human-readable first** — Everything is markdown.
2. **Simplicity** — Flat files, no database, no framework.
3. **Local-first** — Background work runs on Mac via Ollama. Zero API cost.
4. **Ethical by design** — Hard guardrails in the soul file. Full audit logging.
5. **Self-evolving** — The system can grow its own capabilities over time.
