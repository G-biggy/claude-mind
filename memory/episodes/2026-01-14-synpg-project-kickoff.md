# SynPG Project Kickoff

**Date:** 2026-01-14
**Significance:** G landed his first significant freelance design engagement. Built and finalized the SynPG proposal from scratch.

## What Happened

G got a freelance gig designing a website for SynPG — a U.S. B2B outsourced accounting company. Client sent a document with website structure and an email specifying highlighted headers and a blog section. G and Sage scoped the full project together.

Key scoping decisions made:
- Blog platform options analyzed (Ghost, Notion+Super.so, Sanity CMS, HTML page). Recommended against Medium/Substack (SEO goes to those platforms, not SynPG's site).
- Decided to merge FAQ + Contact into one page (cleaner, standard B2B pattern).
- Success Stories is only 4 bullet points — not enough for its own page, folded into Home.
- Dev team (client's tech team) handles all website development. G delivers design files + documentation.
- Proposal written as modular services: Logo, Website Design, Website Development, SEO, Blog Setup, Contact Form Setup.

Proposal finalized and sent. All three packages approved: Logo (JOD 200), Website Design (JOD 400), SEO (JOD 100).
