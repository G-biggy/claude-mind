# CV Optimizer — Origin Meeting with Anton

**Date:** 2025-12-29 (memory mined 2026-03-03)
**Significance:** First documented CV optimizer planning session. G and Anton (CC) define core product.

## What Happened

G and Anton had a meeting where they worked through the CV optimizer concept. G's notes were added to Notion. Key decisions made:

**Product direction:**
- Input: master CV + job description
- Output: adjusted CV with "fit areas" and "potential gaps" analysis
- Anton suggested pre-optimization step: AI improves master CV *before* job-specific optimization (full auto or user-selectable suggestions)

**Pricing:** $1 per page, top-up credit system. Simple, transparent, motivates concise CVs.

**Dashboard:** Small dashboard showing money spent, CVs optimized, pages, JDs processed.

**Future features deferred:**
- History/CV archive (possible paid add-on)
- Gmail/MCP integration for tracking which CV versions got responses
- A/B testing CV versions
- Role-specific fine-tuning

**Technical:** Backend already functional using Supabase (~$50/month hosting).
