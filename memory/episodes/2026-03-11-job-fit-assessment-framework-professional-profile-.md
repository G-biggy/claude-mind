# Job Fit Assessment Framework & Professional Profile System

## What Happened

A comprehensive conversation that resulted in:
1. **Role assessment:** Temenos "Lead Product Designer - Design Partner Program" evaluated and marked as SKIP (Yellow trending Red)
2. **Three-tier framework created:** Category A (Deep Experience), Category B (Learnable Fast), Category C (Deal-Breakers)
3. **Professional profile extracted:** Complete 26-year career history, 6 portfolio projects, work style, philosophy
4. **Three persistent storage layers built:**
   - Local markdown: `g-professional-profile.md` for CLI access
   - Notion page: Mobile-friendly reference
   - Job-fit-assessment skill: References both above + assessment framework

## Key Decisions Made

### Temenos Role: SKIP Decision
**Why:** Role optimizes for external customer co-design facilitation + design thinking evangelism. G's strength is internal strategy + product execution. While G *could* learn this, it's not authentic to his zone.

**The honest assessment:** Better to skip misaligned roles than land one and struggle in interviews. Credibility > fit.

### Professional Profile as Foundation
Everything from this point forward uses the professional profile as the source of truth:
- What G has actually done (Category A)
- What he can learn (Category B)
- What he can't fake (Category C)

### Job-Fit-Assessment Skill Purpose
Quick go/no-go decision tool before CV tailoring. Prevents wasting time on misaligned roles. Gets smarter as we assess more roles.
