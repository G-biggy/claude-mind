# SynPG Logo Direction & Brand System

**Date:** 2026-01-15
**Significance:** Logo concept selected. Full brand system defined and delivered.

## What Happened

G and Sage researched B2B accounting color theory, built 3 logo concepts as interactive React artifacts, and presented them to the client via a Figma deck.

**Three concepts explored:**
- Smart — minimalist wordmark, negative space
- Growth — upward arrow as letter G, wordmark
- Period — Deloitte-inspired wordmark with accent element

**Client selected:** Growth concept, Orange variant (#E07A3D)

## Outcome

Full brand system built from the selection:
- Black (#1a1a1a) + Orange (#E07A3D) primary palette
- Montserrat font family (Google Fonts, free)
- Full neutral scale + semantic colors
- Typography scale: H1 36px Bold through Caption 12px Medium
- Brand guidelines delivered as HTML file for developer reference

## Figma Deck
Presentation structure: section dividers (text bottom-left) + content slides (text top-left). Client reviewed live and gave feedback via conversation. Sage provided text and ordering suggestions.
