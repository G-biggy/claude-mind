**Geographic openness in Sept 2025.** G was open to Dubai, Zurich, Bangkok relocation, and multiple remote setups. By March 2026 he's based in Amman - not sure if that shift represents a preference change or circumstance.

**The master CV creation conversation** reveals he had at least 3 prior CV versions before creating the master. The master was built by synthesizing all versions plus the case study metrics.
